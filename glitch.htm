FAQ

Can I use this in *whatever*?

Yep, I'd like you to link back to me or this page if you could, but if you really don't want to then it's fine.

Does this use jQuery?

No, this only uses default vanilla Javascript.

Why is this HTML5?

This uses the box-shadow and text-shadow CSS rule, only used in CSS 3 (HTML5).

This looks cool, where can I learn Javascript to use it?

If you can build a webpage you can probably use this, it really is that simple. If you really want to learn Javascript then there are a lot of books out there and websites that can help you. I prefer to learn by looking at sample code and working it out that way, hence why I wouldn't call myself a Javascript developer, I'm more of a developer who can use Javascript.
