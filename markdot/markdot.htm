Markdot

Making the web simple for writers, and powerful for designers.

Simple

The project is based around the idea that anybody, without any training or experience, can edit and maintain the contents of a website without having to hassle their web designer to change it for them. The idea is simple, a single text document hosted somewhere that the owner of the site can change at any time.

Modern

The project is written in Javascript, and creates a full HTML5 page on load, allowing for element manipulation via Javascript, and complete customisation through CSS. There are no limitations.

Smart

Quite Smart

The project is written to understand plain text with a basic format, it automatically correctly adds hyperlinks and images from the text, and attempts to correctly format the headings with the idea that headings do not end in a dot, or any other type of punctuation.

Really Smart

Markdot also supports subheaders based off of how English works.

Awesome
